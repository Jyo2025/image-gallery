# 📷 Photo Gallery with Animated Image Swap

This is a simple and interactive **Photo Gallery Web App** built using **HTML, CSS, and JavaScript**.

It features a **main image display** with a **row of thumbnails below**. When any thumbnail is clicked, it **swaps with the main image using a smooth animation**, creating a visually pleasing and interactive user experience.

---

## 🔧 Features

- ✅ Clickable thumbnail images
- ✅ Smooth animated swap effect using JavaScript & `transform`
- ✅ Responsive layout using Flexbox
- ✅ Clean and modern UI with hover effects

---

## 💡 How It Works

- The gallery displays a **main image** at the top.
- A row of **thumbnail images** is displayed below.
- On clicking a thumbnail:
  - It **animates and moves into the main image position**.
  - The main image **moves down into the thumbnail's position**.
  - Behind the scenes, the image `src` values are swapped.

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the gallery
- **CSS3** – Styling, transitions, and layout
- **JavaScript** – DOM manipulation and swap animations

---
